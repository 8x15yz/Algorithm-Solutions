r, e, v 문자 뒤에 오는 숫자들을 모두 더해서 형식에 맞게 재구성하는 작업 조건으로: 1. rev 뒤에는 2개까지 올 수 있음 (ex. r10) 2. 뒤에 2개까지 오지만, 11 이상부터는 무효. 11은 그냥 1으로 수집 
def solution(data):
                ## 1차시도
                res = 0
                for a in range(len(data)):
                    if data[a] in 'rev':
                        if data[a+1:a+2] in 'rev':
                            continue
                        if data[a+1:a+3].isdigit() and 10 >= int(data[a+1:a+3]):
                            res += int(data[a+1:a+3])
                        else:
                            res += int(data[a+1:a+2])
                res = str(res)[0] + '월 ' + str(res)[1:] + '일'
                return res

01 정확성

⚠️ 문제 있음 — ValueError 발생 가능

data[a]가 r/e/v이고 다음 문자가 r/e/v도, 숫자도 아닐 때 else에서 int(data[a+1:a+2])가 터진다. 예: "reverse"에서 'r' 다음 's' → int('s') → ValueError. 또한 '' in 'rev'가 True라 끝 문자 처리가 의도와 다를 수 있다.

04 코드 품질

⚠️ 문제 있음 — 3가지

① 변수명: a → i, res → total
② data[a] in 'rev': 빈 문자열도 포함되어 오해 유발 → data[i] in {'r','e','v'}
③ 슬라이스 중복: one, two 변수로 분리 

def solution(data):
    total = 0
    for i in range(len(data)):
        if data[i] not in {'r', 'e', 'v'}:
            continue
        one = data[i+1:i+2]
        if not one.isdigit():
            continue
        two = data[i+1:i+3]
        if two.isdigit() and int(two) <= 10:
            total += int(two)
        else:
            total += int(one)
    s = str(total)
    return s[0] + '월 ' + s[1:] + '일'
